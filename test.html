Excel Cheat Sheet for Advanced Functions

XLOOKUP

XLOOKUP(lookup_value, lookup_array, return_array, [if_not_found], [match_mode], [search_mode])

  • Purpose: Searches a range or an array, and returns an item corresponding to the first match it finds. If a match doesn't exist, it can return an optional default value.
  • Example: =XLOOKUP("Apple", A2:A5, B2:B5, "Not Found") looks for "Apple" in the range A2:A5 and returns the corresponding value from B2:B5.

COUNTIFS

COUNTIFS(criteria_range1, criteria1, [criteria_range2, criteria2], ...)

  • Purpose: Counts the number of cells that meet multiple criteria across multiple ranges.
  • Example: =COUNTIFS(A2:A5, ">10", B2:B5, "<20") counts cells where values in A2:A5 are greater than 10 and corresponding values in B2:B5 are less than 20.

SUMIFS

SUMIFS(sum_range, criteria_range1, criteria1, [criteria_range2, criteria2], ...)

  • Purpose: Sums cells in a range that meet multiple criteria across multiple ranges.
  • Example: =SUMIFS(B2:B5, A2:A5, ">10", C2:C5, "<20") sums values in B2:B5 where corresponding values in A2:A5 are greater than 10 and in C2:C5 are less than 20.

AND/OR

  • AND(logical1, [logical2], ...)
    • Purpose: Returns TRUE if all arguments are TRUE.
    • Example: =AND(A2>10, B2<5) returns TRUE if A2 is greater than 10 and B2 is less than 5.
  • OR(logical1, [logical2], ...)
    • Purpose: Returns TRUE if any arguments are TRUE.
    • Example: =OR(A2>10, B2<5) returns TRUE if either A2 is greater than 10 or B2 is less than 5.

Tables

  • Creating a Table: Select your data range and press Ctrl + T. This converts the range into a table, offering features like sorting, filtering, and structured references.
  • Structured Reference: When you use formulas within a table, Excel uses structured references that are easier to understand. For example, =SUM(Table1[Sales]) sums all the values in the "Sales" column of "Table1".

Slicers

  • Purpose: Slicers are visual filters for tables, pivot tables, and pivot charts. They allow for a quick and easy way to filter data.
  • Creating a Slicer: Click anywhere inside your table, go to the Table Design tab, and select Insert Slicer. Choose the column you want to filter by.
  • Usage: Click on the items in the slicer to filter your table or pivot table data based on those items.
